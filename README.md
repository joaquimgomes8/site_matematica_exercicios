# 🧮 Exercícios de Matemática

Um site para praticar matemática de forma simples e divertida. A cada rodada é gerada uma conta aleatória, e o objetivo do usuário é responder corretamente o resultado.

## 🌐 Acesse o projeto

👉 https://matematica-exercicios.onrender.com/

## 📖 Sobre o projeto

O projeto foi desenvolvido para treinar lógica de programação e manipulação do DOM utilizando JavaScript.

Sempre que uma nova questão é gerada, o sistema cria automaticamente uma operação matemática aleatória. O usuário deve informar a resposta e o sistema verifica se ela está correta, fornecendo feedback imediato.

## ✨ Funcionalidades

- ✅ Geração aleatória de contas matemáticas
- ➕ Operações matemáticas automáticas
- 🎲 Exercícios diferentes a cada tentativa
- ✔️ Validação da resposta do usuário
- ⚡ Feedback instantâneo
- 📱 Interface simples e responsiva

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
